# Christensen Custom Fabrications Website

This site has been built from a professional client-attraction blueprint:

- Mobile welding
- Custom fabrication
- Residential welding
- Emergency field service

## Files

- `index.html`: Complete one-page site with section anchors and service details
- `styles.css`: Visual design system (color palette, typography, layout)
- `README.md`: This guide

## Deploy

1. Open `index.html` in a browser.
2. Update the placeholders for region and contact details:
   - `[Your City]`
   - `[Your Region; update in copy]`
   - Phone and email in the contact section
3. Replace the hero and project image URL with owned project photos.

## Notes

- The theme is customer-centric and outcome-focused.
- SEO headline and meta description are already set in `index.html`.
- For multi-page expansion, duplicate service block sections into separate files and adjust navigation links accordingly.
